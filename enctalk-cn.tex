\begin{exampleblock}{计算机如何读取“字符”？}\pause
\begin{itemize}%[<+->]
\item 计算机只能读取 0 和 1；
\item 计算机只能读取数字；
\item 将字符转写为数字；% 0 和 1 的序列；
\item 建立一个字符到数字的映射！
\end{itemize}
\end{exampleblock}
